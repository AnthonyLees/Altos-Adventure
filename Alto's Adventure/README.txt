Project Name: Alto's Adventure
Project Description: This game brings the popular mobile app Alto's Adventure to life using the knowledge learned in 15-112. In essence,
this game revolves around a player skiing down a hill, having to avoid obstacles and collect items by jumping and flipping, and
essentially going for as long as possible on a randomly generated track.
How to run the game: You simply need run the main python file called "Term Project" in order to run the game.
Libraries that need to be installed: The PIP library and pygame libraries must be installed
Shortcuts: No shortcuts are necessary to show the features of this game.
